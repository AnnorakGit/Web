# Annorak Intelligence Group - Website

Este documento describe el desarrollo de la página web corporativa para Annorak Intelligence Group.

## Objetivo del Proyecto

Crear una página web de una sola página (single-page) con un diseño moderno, minimalista y sofisticado. La web servirá como carta de presentación de la empresa, mostrando sus servicios, proyectos, visión y datos de contacto.

## Diseño y Estética

La estética está inspirada en la interfaz de Notion: limpia, espaciosa y elegante. Se utilizarán elementos visuales modernos como videos de fondo, animaciones sutiles e interacciones dinámicas para ofrecer una experiencia de usuario fluida y atractiva.

## Estructura de la Página

1.  **Header:** Fijo, con logo y menú de navegación.
2.  **Hero Section:** Sección principal con un video de fondo y el eslogan de la empresa.
3.  **Services:** Grid con los servicios ofrecidos.
4.  **Featured Projects:** Scroll horizontal con videos de proyectos destacados.
5.  **About:** Sección de dos columnas con información sobre la empresa.
6.  **Testimonials:** Slider con testimonios de clientes.
7.  **Contact:** Formulario de contacto e información directa.
8.  **Footer:** Logo, redes sociales y texto legal.

## Tecnología

-   **Framework:** React
-   **Build Tool:** Vite
-   **Hosting:** Vercel

Este documento se mantendrá actualizado a medida que el proyecto avance.
